1. Remove {% section %} tags 2. Move surrounding code into their own sections 3. Replace the Liquid template with a JSON template 4. Add in references to the sections into your new JSON template Example: 1. Copy code from: templates/404.liquid, which is:

{{ 'general.404.title' | t }}

{{ 'general.404.subtext' | t }}

{{ general.404.link | t }}

2. Create: sections/404-template (automatically generates a schema) and then call it 404 template for example. {% schema %} { "name": "404 template", "settings": [] } {% endschema %} {% stylesheet %} {% endstylesheet %} {% javascript %} {% endjavascript %} Paste code from step templates/404.liquid into sections/404-template (above the schema), so now it looks like...

{{ 'general.404.title' | t }}

{{ 'general.404.subtext' | t }}

{{ general.404.link | t }}

{% schema %} { "name": "404 template", "settings": [] } {% endschema %} {% stylesheet %} {% endstylesheet %} {% javascript %} {% endjavascript %} 3. Replace the Liquid template with a JSON template by a: Delete templates/404.liquid b: Create new template called 404 and select JSON as file type. This automatically generates boilerplate JSON. Modify it to look as: { "sections": { "main": { "type": "404-template", "settings": { } } }, "order": [ "main" ] } Video times of above example 1. https://youtu.be/DO9JUW-6WN8?t=370 2. https://youtu.be/DO9JUW-6WN8?t=403 3. https://youtu.be/DO9JUW-6WN8?t=444
